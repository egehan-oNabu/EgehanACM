## 🧠 oNabu — AI Agile Coach

> **Turning team signals into actionable intelligence.**

oNabu is an AI-powered Agile Coaching platform that integrates across your entire dev stack — Slack, Jira, Google Workspace, Microsoft Teams — and surfaces insights your team can actually act on.

<p align="center">
  <a href="https://onabu.ai">
    <img src="https://img.shields.io/badge/-→%20onabu.ai-6C47FF?style=for-the-badge" />
  </a>
</p>

| What it does | How |
|---|---|
| 📊 Team insights & analytics | Real-time data from every tool |
| 🤖 AI-generated coaching | LLM agents trained on Agile methodology |
| 🎙️ Meeting transcription & insights | Automated summaries + action items |
| 🔗 Deep cross-tool integrations | Slack · Jira · Google · Teams |
| 📈 Product analytics pipelines | Automated, zero-config setup |

## 🚀 Currently Building

- **AI Agents for Developer Intelligence** — context-aware agents that understand team velocity, blockers, and patterns
- **Multi-platform integrations** — unified data layer across Slack, Jira, Google Workspace
- **Meeting insights engine** — transcription → summary → action items, fully automated
- **Automated analytics pipelines** — shipping product metrics without manual setup
- **Flutter → Jaspr migration** — moving to HTML-first architecture for better performance and SEO
